USB virus immunity script and hidden files revealer.
DOES NOT REMOVE VIRUSES. MUST BE USED IN CONJUNCTION WITH AN ANTIVIRUS PROGRAM.

1. Download the script file (usb_vaccine.cmd).
2. Unplug all of your USB flash drives, so that if your USB flash drives are
   infected they won't interfere with your computer.
3. Run usb_vaccine.cmd the first time. It will disable the AutoRun and remove
   autorun.inf files from your hard drive.
4. If your Windows has other user accounts, log on to other users and run
   usb_vaccine.cmd FOR EVERY USER. This makes sure the MountPoints2 caches are
   all cleaned.
5. Plug in your USB drives again.
6. Run usb_vaccine.cmd again. This is the last time and will remove
   autorun.inf files from your USB drives. If some of the files have been
   hidden due to the infection, they will be all revealed.
7. Scan your computer with anti-virus software. Your computer is immune now
   but the viruses still exist. You need anti-virus software to remove them.

USB 病毒免疫暨隱藏檔現形程式
**注意：不會真正移除病毒，請搭配防毒軟體一起使用**

1. 下載腳本檔 (usb_vaccine.cmd)
2. 拔除所有的 USB 隨身碟。這樣如果 USB 隨身碟受到感染，你的電腦就不會受隨身碟
   的影響。
3. 執行第一次 usb_vaccine.cmd。此次會關閉 AutoRun 並且移除硬碟上的 autorun.inf
   檔案。
4. 如果你的 Windows 有其他的使用者帳戶，請登入其他的帳戶並在 **每一位使用者**
   上面執行一次 usb_vaccine.cmd。此步驟確保全部的 MountPoints2 快取都清空。
5. 插回去 USB 隨身碟。
6. 再次執行 usb_vaccine.cmd，這是最後一次。此次會清除隨身碟上的 autorun.inf 檔
   案，同時如果有檔案被病毒隱藏了，它們會全部顯示出來。
7. 使用防毒軟體來作一次掃毒。你的電腦現在已經免疫了，但病毒仍存在，要使用防毒
   軟體來清除它們。
